# Contributing

Thanks for your interest in improving the E-Cell SRM Ramapuram landing page.

## How to Contribute

1. Fork the repository and create a new branch.
2. Install dependencies with `npm install`.
3. Make your changes with clear, focused commits.
4. Run `npm run lint` before opening a pull request.
5. Open a pull request with a short summary and screenshots if you changed UI.

## What to Change

- Content updates belong in app/page.tsx.
- Global styling lives in app/globals.css.
- Keep sections modular and readable.

## Reporting Issues

If you find a bug or want to request an enhancement, open a GitHub issue with steps to reproduce and expected behavior.
